# CleanStreet – Smart Street Cleanliness Tracker

## Overview
CleanStreet is developed as part of the **Infosys Springboard Virtual Internship 6.0**.

The goal is to build a smart solution to **report, track, and manage street cleanliness issues** using modern web technologies.

## Tech Stack
- **Frontend:** React + Vite, TailwindCSS
- **Backend:** Node.js + Express
- **Database:** MongoDB

## Getting Started

### Prerequisites
Make sure you have installed:
- **Node.js:** v22.12.0
- **npm:** (comes with Node.js)

---

## Setup & Run

### Frontend (React + Vite)
```bash
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

### Backend (Node.js + Express)
```bash
cd backend

# Install dependencies
npm install

# Start development server
npm run dev
```

## Notes
- Ensure **MongoDB** is running before starting the backend.
- Add a `.env` file in the backend (if your project requires it) for items like `MONGO_URI` and `PORT`.
- After starting the frontend, open the URL shown in the terminal (usually `http://localhost:5173`).
